# 什么是经济？

• “经济”一词，在现代汉语中至少有两个定义

• 一个定义是指节省、有效率，以较少的人力、物力、财力、时间等资源耗费获得较

大的成果。

• 另一定义则用来统称人类社会生产、消费、交换等活动，及其组织这些活动的制度、系统，如国民经济、工业经济、计划经济、市场经济等。
